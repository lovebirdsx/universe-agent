## BaseSandbox（抽象基类）

**文件**：`src/backends/sandbox.ts`

定义了所有沙箱后端必须实现的接口，在 `BackendProtocolV2` 基础上增加了：
- `execute(command)`：Shell 命令执行
- `id`：沙箱唯一标识符
- 生命周期管理（可选）：`initialize`, `cleanup`, `list`, `getOrCreate`, `delete`
- 文件传输（可选）：`downloadFiles`, `uploadFiles`

```typescript
abstract class BaseSandbox implements SandboxBackendProtocolV2 {
  abstract readonly id: string;

  // 必须实现：继承 BackendProtocolV2 的全部方法
  abstract ls(path: string): Promise<LsResult>;
  abstract read(filePath: string, offset?: number, limit?: number): Promise<ReadResult>;
  abstract readRaw(filePath: string): Promise<ReadRawResult>;
  abstract write(filePath: string, content: string): Promise<WriteResult>;
  abstract edit(filePath, oldString, newString, replaceAll?): Promise<EditResult>;
  abstract glob(pattern: string, path?: string): Promise<GlobResult>;
  abstract grep(pattern: string, path?: string, glob?: string | null): Promise<GrepResult>;
  abstract execute(command: string): Promise<ExecuteResponse>;

  // 可选实现：沙箱生命周期
  async initialize(): Promise<void> {}
  async cleanup(): Promise<void> {}
  async list(options?: SandboxListOptions): Promise<SandboxListResponse> {
    return { items: [], cursor: null };
  }
}
```

## LangSmithSandbox

**文件**：`src/backends/langsmith.ts`

### 定位

连接 LangSmith 提供的云端隔离沙箱环境，适用于：
- 需要安全隔离的代码执行
- 不想在本地环境执行危险命令
- 集成 LangSmith 监控和可观测性

### 构造参数

```typescript
interface LangSmithSandboxOptions {
  sandboxId?: string;     // 复用已有沙箱，不填则创建新的
  apiUrl?: string;        // LangSmith API URL（默认读取环境变量）
  apiKey?: string;        // LangSmith API Key（默认读取环境变量）
  metadata?: Record<string, string>;  // 自定义元数据
}
```

### 工作原理

```typescript
class LangSmithSandbox extends BaseSandbox {
  async execute(command: string): Promise<ExecuteResponse> {
    // 通过 LangSmith HTTP API 在远程沙箱执行命令
    const response = await fetch(`${this.apiUrl}/sandboxes/${this.id}/execute`, {
      method: "POST",
      headers: { "X-API-Key": this.apiKey },
      body: JSON.stringify({ command }),
    });

    const result = await response.json();
    return {
      output: result.stdout + result.stderr,
      exitCode: result.exitCode,
      truncated: result.truncated,
    };
  }

  async write(filePath: string, content: string): Promise<WriteResult> {
    // 通过 LangSmith API 写入沙箱文件系统
    await this.uploadFiles([{
      path: filePath,
      content: Buffer.from(content),
    }]);
    return { path: filePath, filesUpdate: null };
  }
}
```

### 使用示例

```typescript
import { LangSmithSandbox } from "universe-agent";

const sandbox = new LangSmithSandbox({
  apiKey: process.env.LANGSMITH_API_KEY,
});

const agent = createUniverseAgent({
  backend: sandbox,
});

// 现在 execute 工具可用，并且在 LangSmith 沙箱中执行
const result = await agent.invoke({
  messages: [{ role: "user", content: "Run the test suite and report results" }]
});
```

## LocalShellBackend

**文件**：`src/backends/local-shell.ts`

### 定位

在本地运行 Shell 命令（无隔离），同时提供完整的文件系统操作能力。

### 特点

- **执行环境**：本机 Shell（bash/sh）
- **文件存储**：本地文件系统（继承 FilesystemBackend）
- **安全性**：无沙箱隔离，命令直接在当前环境执行
- **超时控制**：可配置命令超时

### 构造参数

```typescript
interface LocalShellBackendOptions {
  rootDir?: string;        // 文件系统根目录（默认 process.cwd()）
  commandTimeout?: number; // 命令超时（毫秒，默认 30000）
  maxOutputSize?: number;  // 最大输出大小（字节，默认 1MB）
}
```

### 实现要点

```typescript
class LocalShellBackend extends FilesystemBackend implements SandboxBackendProtocolV2 {
  readonly id = "local-shell";

  async execute(command: string): Promise<ExecuteResponse> {
    try {
      const { stdout, stderr } = await execAsync(command, {
        timeout: this.commandTimeout,
        maxBuffer: this.maxOutputSize,
        cwd: this.rootDir,
      });

      const output = (stdout + stderr).trim();
      return {
        output: output || "(no output)",
        exitCode: 0,
        truncated: false,
      };
    } catch (error: any) {
      return {
        output: error.stderr || error.message,
        exitCode: error.code ?? 1,
        truncated: error.signal === "SIGTERM",  // 超时被截断
      };
    }
  }
}
```

### 使用示例

```typescript
import { LocalShellBackend } from "universe-agent";

const agent = createUniverseAgent({
  backend: new LocalShellBackend({
    rootDir: process.cwd(),
    commandTimeout: 60000,  // 60s 超时
  }),
});

// execute 工具现在可以运行本地命令
// 例如：npm test、python script.py 等
```

## 三种执行后端对比

| 特性         | LangSmithSandbox | LocalShellBackend | StateBackend |
| ------------ | ---------------- | ----------------- | ------------ |
| execute 支持 | ✅ 远程           | ✅ 本地            | ❌            |
| 隔离性       | 完全隔离         | 无隔离            | N/A          |
| 网络访问     | 沙箱内           | 完整              | N/A          |
| 持久化       | 会话级           | 磁盘              | Checkpoint   |
| 适用场景     | 生产安全执行     | 开发环境          | 无需执行     |

```typescript
// 文件格式判断
function isFileDataV1(data: FileData): data is FileDataV1 {
  return Array.isArray(data.content);
}

// MIME 类型检测
function getMimeType(filePath: string): string {
  // 根据文件扩展名返回 MIME 类型
  const ext = path.extname(filePath).toLowerCase();
  return MIME_TYPES[ext] ?? "application/octet-stream";
}

function isTextMimeType(mimeType: string): boolean {
  return (
    mimeType.startsWith("text/") ||
    mimeType === "application/json" ||
    mimeType === "application/xml" ||
    mimeType === "application/javascript"
  );
}

// 内容格式化
function formatContentWithLineNumbers(
  content: string | string[],
  startLine: number,
): string {
  const lines = Array.isArray(content) ? content : content.split("\n");
  return lines
    .map((line, i) => `${String(startLine + i).padStart(4)} ${line}`)
    .join("\n");
}

// 工具调用 ID 清理（用于文件路径）
function sanitizeToolCallId(id: string): string {
  return id.replace(/[^a-zA-Z0-9_-]/g, "_").slice(0, 64);
}

// 超长输出截断
function truncateIfTooLong(
  lines: string[],
  maxLines: number = 200,
): string | string[] {
  if (lines.length <= maxLines) return lines;
  return lines.slice(0, maxLines).join("\n") + `\n[... ${lines.length - maxLines} more lines truncated]`;
}

// V1 → V2 格式转换
export const MAX_LINE_LENGTH = 5000;
```
