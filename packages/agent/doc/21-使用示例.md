# 使用示例

---

## 最简用法

```typescript
import { createDeepAgent } from "deepagents";

const agent = createDeepAgent();

const result = await agent.invoke({
  messages: [
    { role: "user", content: "请分析这个项目的代码结构并生成一份报告" }
  ]
});

console.log(result.messages.at(-1)?.content);
```

默认行为：
- 使用 `claude-sonnet-4-6` 模型
- StateBackend（文件存储在内存状态中）
- 包含所有内置工具（ls、read_file、write_file、edit_file、glob、grep、task、write_todos）
- 包含通用子代理

## 自定义模型

```typescript
import { createDeepAgent } from "deepagents";
import { ChatOpenAI } from "@langchain/openai";

// 方法1：字符串格式
const agent1 = createDeepAgent({
  model: "openai:gpt-4o",
});

// 方法2：模型实例
const agent2 = createDeepAgent({
  model: new ChatOpenAI({ model: "gpt-4o" }),
});

// 方法3：Anthropic（默认，自动启用提示缓存）
const agent3 = createDeepAgent({
  model: "anthropic:claude-opus-4-6",
});
```

## 使用文件系统后端

```typescript
import { createDeepAgent, FilesystemBackend } from "deepagents";

const agent = createDeepAgent({
  backend: new FilesystemBackend({
    rootDir: process.cwd(),  // 工作目录
  }),
});

// Agent 现在可以读写当前目录的文件
const result = await agent.invoke({
  messages: [{ role: "user", content: "读取 package.json 并告诉我项目名称" }]
});
```

## 带结构化输出

```typescript
import { createDeepAgent } from "deepagents";
import { ToolStrategy } from "langchain";
import { z } from "zod";

const ResponseSchema = z.object({
  summary: z.string().describe("简要摘要"),
  issues: z.array(z.string()).describe("发现的问题列表"),
  score: z.number().min(0).max(10).describe("代码质量评分"),
});

const agent = createDeepAgent({
  responseFormat: ToolStrategy.fromSchema(ResponseSchema),
});

const result = await agent.invoke({
  messages: [{ role: "user", content: "审查这段代码的质量" }]
});

// 类型安全的结构化输出
const { summary, issues, score } = result.structuredResponse;
```

import { createDeepAgent } from "deepagents";

const dataAnalyzer = createDeepAgent({

      runnable: dataAnalyzer,  // 使用已编译的 Agent
    }

## 自定义中间件

```typescript
import { createDeepAgent } from "deepagents";
import { createMiddleware, tool } from "langchain";
import { z } from "zod";

// 创建自定义中间件
const WeatherMiddleware = createMiddleware({
  name: "WeatherMiddleware",
  stateSchema: z.object({
    lastWeatherQuery: z.string().optional(),
  }),
  tools: [
    tool(
      async ({ location }) => {
        // 调用天气 API
        return `${location}: 晴天，25°C`;
      },
      {
        name: "get_weather",
        description: "获取指定城市的当前天气",
        schema: z.object({
          location: z.string().describe("城市名称"),
        }),
      }
    )
  ],
});

const agent = createDeepAgent({
  middleware: [WeatherMiddleware],
});

const result = await agent.invoke({
  messages: [{ role: "user", content: "北京今天天气怎么样？" }]
});

// 访问自定义状态（TypeScript 自动推断类型）
console.log(result.lastWeatherQuery);  // string | undefined
```

## 加载记忆（AGENTS.md）

```typescript
import { createDeepAgent, FilesystemBackend } from "deepagents";

const agent = createDeepAgent({
  backend: new FilesystemBackend({ rootDir: "/" }),
  memory: [
    "/home/user/.deepagents/AGENTS.md",   // 用户全局规则
    "/projects/my-app/.deepagents/AGENTS.md", // 项目规则
  ],
});
```

示例 AGENTS.md 内容：
```markdown
# 项目规则

## 代码规范
- 使用 TypeScript，不使用 JavaScript
- 所有函数必须有 JSDoc 注释
- 测试覆盖率要求 > 80%

- `npm run build`：编译 TypeScript

- `npm run lint`：运行 ESLint

## 架构约定
- 业务逻辑放在 `src/services/`
- API 路由放在 `src/routes/`
```

## 人工审批（Human-in-the-Loop）

```typescript
import { createDeepAgent } from "deepagents";
import { MemorySaver } from "@langchain/langgraph";

const agent = createDeepAgent({
  checkpointer: new MemorySaver(),  // 需要 checkpointer 支持中断恢复
  interruptOn: {
    "write_file": true,   // 写文件前需要人工确认
    "execute": {
      condition: (args) => args.command.includes("rm"),  // 只对 rm 命令要求确认
    },
  },
});

// 启动对话（会在写文件前暂停）
const threadId = "thread-123";
const result = await agent.invoke(
  { messages: [{ role: "user", content: "删除临时文件并重建项目" }] },
  { configurable: { thread_id: threadId } }
);

// result.interrupted 为 true 时，显示待审批的工具调用
if (result.interrupted) {
  console.log("待审批工具调用：", result.pendingToolCalls);

  // 用户确认后恢复
  const finalResult = await agent.invoke(
    { messages: [], approve: true },
    { configurable: { thread_id: threadId } }
  );
}
```

## 状态持久化

```typescript
import { createDeepAgent } from "deepagents";
import { PostgresSaver } from "@langchain/langgraph-checkpoint-postgres";

const checkpointer = PostgresSaver.fromConnString(
  process.env.DATABASE_URL
);

const agent = createDeepAgent({
  checkpointer,
  name: "my-agent",
});

// 同一 thread_id 的调用共享状态
const threadId = "user-session-456";

// 第一次调用
await agent.invoke(
  { messages: [{ role: "user", content: "帮我创建一个 React 组件" }] },
  { configurable: { thread_id: threadId } }
);

// 后续调用（自动恢复历史状态）
await agent.invoke(
  { messages: [{ role: "user", content: "给组件添加 TypeScript 类型" }] },
  { configurable: { thread_id: threadId } }
);
```

## 使用异步子代理

```typescript
import { createDeepAgent } from "deepagents";

const agent = createDeepAgent({
  subagents: [
    {
      name: "data-processor",
      description: "长时间运行的数据处理代理（异步）",
      graphId: "data-processor-graph",        // 远程 LangGraph 图 ID
      url: "https://langgraph-server.com",    // Agent Protocol 服务器
      headers: {
        "Authorization": `Bearer ${process.env.API_TOKEN}`,
      },
    }
  ]
});

// LLM 会自动使用 create_async_task / check_async_task 工具
// 不阻塞主线程，可以继续做其他工作
```

## 流式输出

```typescript
import { createDeepAgent } from "deepagents";

const agent = createDeepAgent();

// 流式处理 Agent 输出
for await (const event of agent.streamEvents(
  { messages: [{ role: "user", content: "分析这个代码库" }] },
  { version: "v2" }
)) {
  if (event.event === "on_chat_model_stream") {
    // 实时显示 LLM 生成的文本
    process.stdout.write(event.data.chunk.content);
  }

  if (event.event === "on_tool_start") {
    console.log(`\n[调用工具: ${event.name}]`);
  }
}
```
