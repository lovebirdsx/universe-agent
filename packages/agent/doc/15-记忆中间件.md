# 记忆中间件

**文件**：`src/middleware/memory.ts`

记忆中间件实现了 [AGENTS.md 规范](https://agentsmd.org)，将 Markdown 格式的"记忆文件"注入到 Agent 的系统提示中，让 Agent 知道项目规范、编码风格等上下文信息。

---

## AGENTS.md 规范

AGENTS.md 是一种约定的文件格式，类似于 `CLAUDE.md`（Claude Code 使用的约定），内容包括：
- 项目特定的指令和规则
- 代码风格指南
- 架构文档
- 构建/测试命令
- 任何 Agent 应该了解的上下文

---

## 工作原理

```
配置记忆文件路径（如 ~/.universe-agent/AGENTS.md）
    │
    ▼
wrapModelCall 钩子触发
    │
    ▼
通过后端读取文件内容
    │
    ▼
将文件内容注入到系统提示末尾
（可选：添加 Anthropic 提示缓存标记）
    │
    ▼
LLM 推理时包含记忆上下文
```

---

## 典型的记忆文件路径

```typescript
createUniverseAgent({
  memory: [
    "~/.universe-agent/AGENTS.md",        // 用户全局记忆
    "./.universe-agent/AGENTS.md",         // 项目根记忆
    "./.universe-agent/team-rules.md",     // 团队规则
  ]
})
```

路径解析：
- `~` 展开为用户主目录
- 相对路径相对于后端根目录
- 读取失败的路径被忽略（不报错）

## 注入格式

```typescript
// wrapModelCall 中的注入逻辑
const memoryBlocks = await Promise.all(
  sources.map(async (source) => {
    const result = await backend.read(source);
    if (result.error) return null;

    const displayName = getDisplayName(source);  // 从路径派生显示名

    return addCacheControl
      ? {
          type: "text",
          text: `\n\n## Memory: ${displayName}\n\n${result.content}`,
          cache_control: { type: "ephemeral" },  // Anthropic 提示缓存
        }
      : `\n\n## Memory: ${displayName}\n\n${result.content}`;
  })
);

const systemMessage = request.systemMessage.concat(memoryBlocks.filter(Boolean));
```

---

## 提示缓存优化

当 `addCacheControl: true`（使用 Anthropic 模型时自动启用）：

```typescript
// 每个记忆块添加 cache_control
{
  type: "text",
  text: "...",
  cache_control: { type: "ephemeral" }  // 告知 Anthropic 缓存此内容块
}
```

记忆内容通常较大且固定，启用缓存可以显著降低 API 调用成本（后续调用不重复计算已缓存 token）。

## 状态管理

```typescript
stateSchema: z.object({
  memoryContents: z.record(z.string(), z.string()).default({})
})
```

`memoryContents` 被排除在子代理状态传递之外，各 Agent 独立加载各自配置的记忆。

---

## 显示名派生规则

从文件路径自动派生显示名，展示在系统提示的标题中：

```
~/.universe-agent/AGENTS.md         → "User Global Memory"
./.universe-agent/AGENTS.md         → "Project Memory"
./.universe-agent/team-rules.md     → "Team Rules"
/custom/path/my-context.md      → "My Context"
```

## MemoryMiddlewareOptions

```typescript
interface MemoryMiddlewareOptions {
  backend: AnyBackendProtocol | BackendFactory;  // 必填
  sources: string[];    // 记忆文件路径列表，必填
  addCacheControl?: boolean;  // 是否添加 Anthropic 缓存标记（默认 false）
}
```
