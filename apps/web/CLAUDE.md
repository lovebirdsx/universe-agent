**核心结构：**

| 文件            | 说明                    |
| --------------- | ----------------------- |
| `index.html`    | HTML 模板               |
| `src/main.tsx`  | React 入口              |
| `src/App.tsx`   | 主组件（产品列表 Demo） |

**注意：**

* 依赖 `@universe-agent/shared`（formatMoney、cn）和 `@universe-agent/ui`（Button）
* 测试环境使用 happy-dom
